# python-study

Pythonの学習

## 環境

- OS：Ubuntu 20.04 LTS (native/WSL2)
- Python 3.11.4
    - [Release](https://www.python.org/downloads/release/python-3114/)
    - インストール
        ```
        $ ./configure
        $ make
        $ make test
        $ sudo make install
        ```
- 仮想環境：venv
    - `$ python -m venv .venv && source .venv/bin/activate`
